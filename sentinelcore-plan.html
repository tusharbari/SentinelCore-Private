Final Year Project Blueprint

SentinelCore

Enterprise Security Operations Platform — a full-stack capstone covering threat detection, incident response, vulnerability management, and compliance automation.

Backend  Java 17 + Spring Boot 3
Frontend  React 18 + Tailwind CSS
Database  PostgreSQL + MongoDB
Auth  Spring Security + JWT
Realtime  WebSocket (STOMP)
Queue  Apache Kafka / RabbitMQ
Cache  Redis
Build  Maven · Vite · Docker

Modules — 18 Core Components

01

Auth & RBAC

JWT-based login with role-based access control — Admin, Analyst, Viewer — plus MFA toggle and session management for secure multi-tenant access.

02

User & Team Management

CRUD for security team members with department assignment, skill tagging, and shift scheduling so incidents route to the right on-call analyst.

03

Security Dashboard

Real-time overview with threat-level gauges, open-incident counters, MTTR trends, and a live event feed powered by WebSocket push.

04

Threat Intelligence Feed

Ingests IOC data (IPs, hashes, domains) from external sources and internal logs, normalizes it into a searchable threat library with severity scores.

05

Incident Management

Create, classify (P1–P4), assign, escalate, and resolve security incidents through a Kanban-style workflow with SLA countdown timers.

06

Alert Engine

Rule-based alerting that correlates log events against detection rules, deduplicates noise, and pushes prioritized alerts to the analyst queue.

07

Vulnerability Scanner

Scans registered assets for known CVEs using a local vulnerability database, scores findings with CVSS, and links them to remediation playbooks.

08

Asset Inventory

Central registry of servers, endpoints, and network devices with OS, owner, criticality tier, and patch status — the backbone for risk scoring.

09

Log Management (SIEM Lite)

Collects and indexes structured logs (syslog, app logs) into MongoDB, with full-text search, time-range filters, and saved-query support.

10

Playbook Automation

Pre-defined and custom response playbooks (e.g., "isolate host → block IP → notify") that analysts can trigger manually or attach to alert rules.

11

Compliance Manager

Maps security controls to frameworks (ISO 27001, NIST, PCI-DSS), tracks evidence collection status, and generates gap-analysis reports.

12

Audit Trail

Immutable log of every user action — logins, config changes, incident updates — stored in an append-only collection for forensic accountability.

13

Report Generator

On-demand and scheduled PDF/CSV reports — executive summaries, incident post-mortems, vulnerability aging, compliance posture — with chart embeds.

14

Notification Service

Multi-channel dispatch (email, in-app toast, webhook) with per-user preference settings, throttle rules, and escalation chains for critical alerts.

15

Risk Scoring Engine

Aggregates vulnerability counts, open incidents, asset criticality, and compliance gaps into a per-asset and org-wide risk score displayed on dashboards.

16

Knowledge Base

Internal wiki for runbooks, post-incident reviews, and detection-rule documentation — searchable, version-tracked, and linked from incidents.

17

Integration Hub

REST API gateway that exposes SentinelCore data to external tools (Slack, Jira, PagerDuty) and accepts inbound webhooks for event ingestion.

18

Settings & Configuration

Global knobs — data-retention policies, alert thresholds, SMTP/webhook config, UI theme — with an admin-only config panel and env-variable overrides.
